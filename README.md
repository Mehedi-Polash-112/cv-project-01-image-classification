# CV Project 01 — Cats vs Dogs Image Classification

## Project Overview
This is my first Computer Vision project.  
In this project, I trained a deep learning model (ResNet18) to classify images into two categories:
- Cats
- Dogs

The model was trained using PyTorch on Google Colab.

---

## What I Learned
- How to load image datasets using ImageFolder
- How to use transfer learning with a pretrained model
- How to train and evaluate a neural network
- How to generate a confusion matrix
- How to test the model on new images

---

## Model Details
- Architecture: ResNet18 (pretrained)
- Framework: PyTorch
- Input Size: 224x224
- Loss Function: CrossEntropyLoss
- Optimizer: Adam

---

## Results
The model achieved good validation accuracy on the dataset.
See the results folder for:
- Confusion Matrix
- Classification Report

---

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Upload your own images to test predictions
